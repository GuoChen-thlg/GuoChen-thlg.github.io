OWL Carousel2

发表于 2020-05-24 | 分类于 文档 ， jQuery | | 热度 ℃
字数统计: 910 | 阅读时长 ≈ 3

Owl Carousel 是一个强大、实用但小巧的 jQuery 幻灯片插件，它具有一下特点：

  • 兼容所有浏览器
  • 支持响应式
  • 支持 CSS3 过渡
  • 支持触摸事件
  • 支持 JSON 及自定义 JSON 格式
  • 支持进度条
  • 支持自定义事件
  • 支持延迟加载
  • 支持自适应高度

Owl Carousel 提供了众多的参数、回调函数及自定义事件（具体请往下浏览），所以它几乎可以满足你的所有要求。
